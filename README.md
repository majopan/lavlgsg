## 🎯 Descripción

**SERVILAV** es una plataforma web profesional diseñada para un servicio técnico especializado en lavadoras con sede en Bogotá, Colombia. El sitio presenta de manera clara y atractiva todos los servicios ofrecidos, información de contacto y la experiencia de más de 15 años en el mercado.

### 🚀 ¿Qué ofrece la aplicación?

- **Landing page informativa** con servicios de reparación de lavadoras
- **Diseño responsive** optimizado para todos los dispositivos
- **Contacto inmediato** vía WhatsApp y llamada telefónica
- **Formulario de contacto** para solicitar presupuestos
- **Información detallada** de servicios, cobertura y horarios

### 💡 Nuestros servicios

- 🔧 Reparación de lavadoras (todas las marcas)
- 🛠️ Mantenimiento preventivo
- 📦 Instalación de lavadoras nuevas y usadas
- 🔩 Cambio de piezas y repuestos originales
- ⚡ Diagnóstico electrónico especializado

## ✨ Características

## 📦 Instalación

### Requisitos previos
- Node.js (v14 o superior)
- npm o yarn

### Pasos de instalación

```bash
# 1. Clonar el repositorio
git clone https://github.com/tu-usuario/servilav.git

# 2. Entrar al directorio
cd servilav

# 3. Instalar dependencias
npm install

# 4. Iniciar el servidor de desarrollo
npm start
